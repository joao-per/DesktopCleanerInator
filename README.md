This tool is designed to organize items in your Desktop and Downloads folders, keeping your workspace tidy and clutter-free. ✨

## 🛠️ Functionality

The Desktop Cleaner Inator has the following features:

1. **File Organization**: Automatically moves files into designated folders based on their file types.
2. **Customization**: Easily modify the script to include additional file types and categories.

## 📚 How to Use

1. Place the executable in a convenient location.
2. Double-click the executable to run the application.
3. Your Desktop and Downloads folders will be organized automatically.
